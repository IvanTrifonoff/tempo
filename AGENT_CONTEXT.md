# Контекст Агента (Agent Context) — tempo.TRFNV

## 🎯 Обзор Проекта
Tempo.TRFNV — это профессиональный музыкальный плеер для тренеров бальных танцев. 
**Текущая версия:** 1.0.46 (Stabilization Edition)

## 🏗 Архитектура
Проект перешел на полностью автоматизированный цикл поставки (CI/CD).

### Бэкенд (Node.js + Express + PostgreSQL)
- **Модульность:** Код разделен на `routes`, `controllers`, `services` и `middleware`.
- **База данных:** PostgreSQL. Схема управляется через `node-pg-migrate`.
- **Логирование:** Winston (JSON). Ошибки в `error.log`.

### Фронтенд (React + TypeScript + Vite)
- **Хуки:** Вся логика плеера и аудио в `hooks/`.
- **Оптимизация:** Gzip сжатие и кэширование статики (1 год).

## 🚀 CI/CD и Деплой (NEW!)
- **Автоматизация:** Деплой происходит автоматически при пуше в ветку `main`.
- **Валидация:** Пайплайн проверяет согласованность версий в `constants.tsx` и `CHANGELOG.md`.
- **Реестр:** Docker-образы хранятся в GitHub Container Registry (GHCR).
- **Синхронизация:** Описание изменений из `CHANGELOG.md` автоматически попадает в БД при деплое.

## 🛑 Важные правила для Агента
1. **Версионность:** Перед пушем в `main` ОБЯЗАТЕЛЬНО обнови версию в `constants.tsx` и добавь описание в `CHANGELOG.md`. Иначе билд упадет.
2. **Не деплой руками:** Используй Git. Ручной деплой архивом — только в экстренных случаях.
3. **Миграции:** Всегда создавай файл миграции при изменении структуры БД.

## 📅 Обновление v1.0.46 (Февраль 2026) — Итоги сессии стабилизации

### ✅ Что было сделано:
1. **Стабилизация Audio:** Полностью удален функционал **Clap Detection**. Это устранило конфликты с микрофоном на iOS, решило проблему «черного экрана» и восстановило высокое качество звука в Bluetooth-колонках.
2. **Безопасность (Hardening):**
   - Внедрена валидация загружаемых файлов в Multer (MIME-типы, лимит 20МБ).
   - Переход на **UUID** для идентификаторов треков и пользователей.
   - Улучшена надежность верификации почты (динамическое определение домена).
3. **Синхронизация Git:** Все файлы бэкенда (`routes`, `controllers`, и т.д.) теперь добавлены в репозиторий. GitHub содержит актуальное состояние проекта.

### ⚠️ Актуальный техдолг для следующего Агента:
1. **Database Consistency:** Проверить синхронность `server/schema.sql` и файлов в `server/migrations/`.
2. **CORS:** Ограничить доступ в `server.js` конкретным доменом `tempo.trfnv.ru`.
3. **Transactions:** Добавить `BEGIN/COMMIT` в критические операции регистрации и загрузки.
4. **Git на VPS:** После настройки Deploy Keys на сервере выполнить `git reset --hard origin/main`, чтобы очистить статус локальных файлов.
