## [1.0.35] — 2025-12-31
### ✨ Новые функции
- **Demo Tracks:** Администраторы теперь могут помечать любые треки как "Демо". Такие треки становятся доступны всем пользователям системы (включая неавторизованных) без необходимости добавления в личную библиотеку.

### 🏗 Рефакторинг и Оптимизация
- **TrackVisibility Service:** Логика проверки прав доступа вынесена в отдельный сервис на бэкенде. Удалено дублирование SQL условий.
- **Unified Visibility:** Поля `is_demo` и `is_public` объединены в единый флаг публичности в базе данных.

## [1.0.34] — 2025-12-31
### 🏗 Архитектурная стабилизация
- **Modular Backend:** Полная декомпозиция `server.js`. Код разбит на контроллеры, роуты и middleware. Это база для будущего расширения.
- **Database Migrations:** Внедрена система миграций `node-pg-migrate`. Теперь изменения структуры БД версионируются и применяются автоматически.
- **Logging 2.0:** Внедрен Winston для структурированного логирования в формате JSON. Ошибки теперь фиксируются в `error.log`.
- **Performance:** Включено Gzip-сжатие на сервере. Время загрузки статики сокращено в 3-4 раза.
- **Cache-Control:** Настроено агрессивное кэширование JS/CSS бандлов (1 год), что делает повторные открытия приложения мгновенными.

### 🛡 Безопасность и Ops
- **Centralized Error Handling:** Все ошибки API обрабатываются единым middleware, предотвращая утечку данных о сервере.
- **Docker Cleanup:** На сервере настроена автоматическая очистка мусора Docker по расписанию.
- **Documentation:** Полное обновление инструкций для ИИ-агентов, актуализирована карта проекта.

## [1.0.33] — 2025-12-31
### 🏗 Архитектура и Производительность
- **PostgreSQL:** Полный переход с JSON-файла на реляционную базу данных.
- **SQL Optimization:** Устранена проблема N+1 запросов при загрузке плейлистов.
- **Refactoring:** Логика аудио-движка и плеера вынесена в кастомные хуки (`usePlayer`, `useAudioContext`).

## [1.0.32] — 2025-12-30
### 🔐 Система ролей и Доступ
- **SaaS Архитектура:** Разделение на роли: Администратор, Тренер и Ученик.
- **Верификация:** Регистрация тренеров с подтверждением через Email.
- **Инвайты:** Ученики регистрируются по уникальной ссылке тренера.
- **PWA:** Исправлена логика обновления приложения.
