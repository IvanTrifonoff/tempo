# Контекст Агента (Agent Context) — tempo.TRFNV

## 🎯 Обзор Проекта
Tempo.TRFNV — это профессиональный музыкальный плеер для тренеров бальных танцев. 
**Текущая версия:** 1.0.47 (Admin & Quality Edition)

## 🏗 Архитектура
Проект перешел на полностью автоматизированный цикл поставки (CI/CD).

### Бэкенд (Node.js + Express + PostgreSQL)
- **Модульность:** Код разделен на `routes`, `controllers`, `services` и `middleware`.
- **Админка:** Роуты `/api/admin/*` защищены `isAdmin` middleware.
- **Логирование:** Winston (JSON). Ошибки в `error.log`.

### Фронтенд (React + TypeScript + Vite)
- **Роутинг:** Используется `react-router-dom` для разделения плеера (`/`) и админки (`/admin`).
- **Стейт:** UI плеера изолирован в `PlayerUI.tsx`, админка в `components/admin/`.

## 🚀 CI/CD и Деплой
- **Автоматизация:** GitHub Actions прогоняет Jest (Backend) и Vitest (Frontend) перед сборкой.
- **Синхронизация:** Описание изменений из `CHANGELOG.md` попадает в БД при деплое.

## 🛑 Важные правила для Агента
1. **Версионность:** Перед пушем в `main` ОБЯЗАТЕЛЬНО обнови версию в `constants.tsx` и `CHANGELOG.md`.
2. **Техдолг и Костыли:** ПЕРЕД РАБОТОЙ ПРОЧИТАЙ [TECH_DEBT_CRUTCHES.md](./docs/TECH_DEBT_CRUTCHES.md). 
3. **Планы:** См. [ROADMAP.md](./docs/ROADMAP.md) для понимания вектора развития.
