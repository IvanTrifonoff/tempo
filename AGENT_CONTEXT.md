## 📍 Обзор проекта
**Tempo.TRFNV** — SaaS-платформа для тренеров бальных танцев.
*   **Стек:** React (Vite) + Node.js (Express) + PostgreSQL.
*   **Ключевые функции:** Плеер с изменением BPM, режим тренера (автопилот, хлопки), управление учениками.

## ⚠️ Технические особенности (Deep Dive)

### 1. Аудио-движок (КРИТИЧНО ДЛЯ iOS)
Проблема iOS Safari: Создание `AudioContext` при загрузке ломает фоновое воспроизведение.
*   **Решение:** Используется ленивая инициализация. `AudioContext` создается только при активации функций тренера (метроном/хлопки) через функцию `initAudioCtx` по клику пользователя.
*   **Фоновый режим:** Работает через `Media Session API` и `latencyHint: 'playback'`.
*   **Стриминг:** Включен `preload="metadata"` для быстрого старта.

### 2. Данные и Модели
*   **Users:** Роли `admin`, `coach`, `student`. Поле `isAdmin` (legacy) сохранено в API для совместимости с фронтендом.
*   **Tracks:** Привязаны к `owner_id`. Публичные треки видны всем, приватные — только тренеру и его ученикам.
*   **Синхронизация:** БД является источником правды. Фронтенд использует JWT для авторизации.

### 3. PWA и Сборка
*   **Tailwind v3:** Локальная сборка. Файл `constants.tsx` должен быть в `content` конфига, иначе PurgeCSS удалит цвета жанров.
*   **PWA:** Режим `autoUpdate`. Проверка обновлений по изменению версии в `constants.tsx` и `localStorage`.
