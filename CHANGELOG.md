# История изменений (Changelog) — tempo.TRFNV

Все заметные изменения в проекте фиксируются в этом файле.

## [1.0.21] — 2025-12-29
### Новые функции
- **Режим тренера:** Отдельная кнопка "Свисток" для доступа к функциям автопилота и детекции хлопков.
- **Настройки:** Отдельное меню (шестеренка) для выбора языка и управления уведомлениями.
- **Уведомления:** Добавлена возможность включить системные Push-уведомления о выходе новых версий.
- **Фоновый режим:** Улучшена работа аудио в фоне на iOS (Media Session API + оптимизация AudioContext).
- **Редактирование:** Администраторы могут редактировать название трека и исполнителя прямо в списке (клик по иконке карандаша).

### Улучшения интерфейса
- **Компактный вид:** Уменьшены отступы в списке треков и хедере для лучшего использования пространства.
- **Плеер:** Добавлен визуальный индикатор Play/Pause на карточку трека при наведении.
- **Плейлисты:** Обновлен дизайн окна создания плейлиста в общем стиле приложения.
- **Иконки:** Обновлены иконки (свисток, карандаш) на более качественные.

### Исправления
- Исправлено прерывание музыки при сворачивании приложения.
- Исправлен ввод текста на экранах входа (проблема iOS).
- Исправлено поведение клика по карточке трека (теперь можно кликать по тексту для воспроизведения).

## [1.0.5] — 2025-12-28
### Улучшено
- **Интерфейс:** Улучшено окно "Что нового" (центровка, контакты поддержки).
- **Система:** Централизованное управление версией приложения.

## [1.0.4] — 2025-12-28
### Улучшено
- **Интерфейс уведомлений:** Улучшен дизайн окна "Что нового" и исправлена локализация.

## [1.0.3] — 2025-12-28
### Исправлено
- **Авторизация:** Исправлена проблема с вводом текста в полях логина и регистрации (удален конфликтующий touch-action).
### Добавлено
- **Уведомления:** Добавлено модальное окно "Что нового" для информирования пользователей о важных обновлениях.

## [1.0.2] — 2025-12-28
### Добавлено
- **PWA Поддержка:** Приложение теперь можно установить на смартфон/ПК.
- **Управление хлопком (v2):** Реализована логика "Двойного хлопка" (Double Clap) для защиты от ложных срабатываний.
- **Выбор аудиоустройств:** Добавлены селекторы для выбора конкретного микрофона (Input) и динамиков (Output).
- **Wake Lock API:** Экран устройства больше не гаснет во время активной тренировки.
- **Индикатор уровня:** Визуальная шкала (VU Meter) громкости микрофона в настройках.
- **Бесшовные обновления:** Настроено фоновое обновление PWA без назойливых уведомлений.

### Исправлено
- Исправлен баг автопилота (сброс времени при переключении трека).
- Унифицирован дизайн переключателей в панели тренера.
- Исправлена логика захвата микрофона (теперь только при включении режима хлопка).

## [1.0.1] — 2025-12-26
### Добавлено
- **Локализация:** Полная поддержка трех языков: Русский, Английский, Испанский.
- **Персональные плейлисты:** Плейлисты теперь привязаны к аккаунту пользователя.
- **Категория «Для детей»:** Новый жанр в библиотеке.
- **Брендинг:** Смена названия на `tempo.TRFNV - dance coach PRO`.

## [1.0.0] — 2025-12-25
### Добавлено
- **Клиент-серверная архитектура:** Переход с SPA на полноценное приложение с бэкендом (Node.js/Express).
- **База данных:** Хранение треков, пользователей и настроек в JSON-БД.
- **Авторизация:** Регистрация и вход с хешированием паролей и JWT-токенами.
- **Загрузка файлов:** Возможность загружать MP3 треки через админ-панель.
- **Docker:** Контейнеризация приложения для стабильного деплоя.
- **HTTPS:** Настройка SSL сертификатов через Certbot.
